## Overview

Official web for **Synapso**, a desktop application focused on cognitive training and performance improvement.

Synapso helps users improve:
- focus and attention  
- reaction time  
- working memory  
- cognitive control  

The platform provides structured exercises with real-time feedback and performance analytics.
